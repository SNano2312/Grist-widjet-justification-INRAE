Table vide
Aucune donnée dans JDP_2025
La table JDP_2025 ne contient aucune ligne pour votre session. Veuillez d'abord importer les données depuis SIFAC dans Grist avant de lancer ce widget.
Erreur — exercice comptable incorrect
Les données ne correspondent pas à 2025
La matrice a détecté une valeur incorrecte dans la colonne exercice. Toutes les lignes doivent contenir 2025.
Vérifiez que vous avez bien utilisé la variante JUSTIF_2025 lors de l'extraction SIFAC, et que vous n'avez pas ouvert ni modifié le fichier d'extraction.
Avertissement — centre de coûts
Valeur de centre de coûts incohérente détectée
Un centre de coûts ne respecte pas le format attendu. Cela peut indiquer que le fichier d'extraction a été ouvert ou modifié avant import.
Les centres de coûts doivent être :
· de 9 ou 10 caractères
· de 8 caractères avec la 5ème lettre = L
· commencer par MSL
· ou appartenir à la liste : C11, C12…C75, COMMUN, INRAE
Les résultats créés par la matrice pourront être erronés si vous forcez l'injection.
